# Backlog archive

Completed backlog items are moved here at sprint end (by `/capture-lessons`,
Step 7) instead of being deleted. Git history would track the deletions, but
archiving keeps them searchable in plain text without scanning logs.

Format for each archived item:

```markdown
## [Sprint XX — YYYY-MM-DD] Item title

[Original item body, preserved verbatim from tasks/backlog.md]
```

This file is **append-only**. Sort by sprint number ascending — oldest at
the top, newest at the bottom. Agents and skills do not read this file by
default; it exists for human searching, postmortems, and pattern detection
across multiple sprints.
